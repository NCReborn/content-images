<details><summary>‚ñ´Ô∏èInstallation Guide</summary>
üîπWelcome to the install guide for this collection.

**IMPORTANT** - We highly recommend you update to the latest version of Expedition 33 and make sure you have [Vortex](https://www.nexusmods.com/site/mods/1?tab=files) installed.

**IMPORTANT**  - Our collections are only compatible with [Vortex](https://www.nexusmods.com/site/mods/1?tab=files).&#x20;

**DISCLAIMER** - Our collections will only work on legitimate copies of the game, purchased through Steam/GoG/Epic stores. Any cracked/illegal copies will not be supported, and you will receive no help diagnosing issues if you have an illegitimate copy, so please don't waste our support teams time if it's not working on a non-legit copy of the game.

üîπ**Vortex Settings**

\- In Settings, go to the Mods section and make sure the Deployment Method is set to Hardlink Deployment. Note: If you encounter issues selecting this option, it might be due to your drive format (e.g., FAT or XFAT). You may need to reformat the drive to a compatible format for hard links.&#x20;

Note: You must also have the location of your staging folder to be in the exact drive your game is installed on, So if your game is installed on your C: drive, then your staging folder must also be on the C: drive for hardlinked to work.

![](https://i.imgur.com/hcbmYvA.png)üîπ**Download the collection**

From our main Nexus page click on add collection, this will open up [Vortex](https://www.nexusmods.com/site/mods/1?tab=files) and begin the installation process. (Make sure to select the latest revision to stay up-to-date with the latest working mods)

![](https://cdn.discordapp.com/attachments/1400950386088607745/1460714785548013578/image.png?ex=696be0a7\&is=696a8f27\&hm=71938a362784d1b59691fdd6c5c25aba79786500f1b504864dfdcd29c3097a05&)üîπ**Vortex: Select profile to install**

Inside of vortex it will now bring a pop-up asking what profile you want to install the collection to. By default this is set to create a new profile (Recommended by Curator), keep this setting as it is and click "Install Now", It may also ask if you wish to switch to that profile, just click yes. (Also make sure "Install mods during collection download" is green/ticked.

![](https://cdn.discordapp.com/attachments/1400950386088607745/1460716265172893901/image.png?ex=696be208\&is=696a9088\&hm=c992989ed458dfee28673bffe3c249b7a8f97112ac458b53c1651fadeae9e4e1&)üîπ**Vortex: Installing mods**

Vortex will now install the mods, if you are a premium users it may only take a couple minutes, if you are a free user, you will have to open each mod page individually and select to add each one to vortex. (If the collection has a small amount of mods this won't take too much longer)

During installation you may run into popups asking which .pak files you wish to install. I should (might forget) have a little popup that appears in the bottom right of vortex with instructions. Just select the options you wish to enable.

üîπ**Vortex: Collection Installed/Optional Mods**

Once the mods have been installed you should get the following pop-up, if any optional mods are present in the revision you are downloading you will be prompted to refuse them, or install them. If you don't want any of the optional mods select "No Thanks", if you wish to add the optional mods click "Install optional mods". Again I should have instructions for each optional mod added and what they change/alter. If you wish to not install it simply skip.

***Note: Please read through each optional mod instructions carefully, as some might make the game easier/harder.***

![](https://cdn.discordapp.com/attachments/1400950386088607745/1460750982907691038/image.png?ex=696c025d\&is=696ab0dd\&hm=523d828a072dc0733549efb7f84e603a84d82d7f9eb9c71bb8edbac55cb2850e&)üîπ**Vortex: Collection Installed/Boot Game**

Congrats, the collection should now be installed correctly, simply deploy the mods, and click the play button at the top of vortex to boot into the game.
